# 🌸 SmartPrompt AI

SmartPrompt AI is an intelligent prompt-based chatbot that generates responses in multiple styles based on the user's selected preference. Instead of providing the same answer every time, it adapts the response to suit different audiences such as beginners, students, interview candidates, or exam preparation.

---

## ✨ Features

- 😊 Simple Mode – Easy-to-understand explanations.
- 📚 Student Mode – Clear explanations for college students.
- 🧠 Detailed Mode – Comprehensive answers with examples.
- 📝 Exam Mode – Point-wise, exam-oriented responses.
- 💼 Interview Mode – Professional interview-style explanations.
- ⚡ Summary Mode – Quick and concise summaries.
- 👶 Explain Like I'm 10 – Child-friendly explanations.
- 🎨 Elegant pastel-themed user interface.
- 🤖 Powered by Google's Gemini AI.
- 📥 Download generated responses.
- 📱 Clean and responsive Streamlit interface.

## ⚙️ How It Works

1. User enters a question.
2. Selects the preferred response style.
3. The application creates a customized prompt.
4. Gemini AI generates the response.
5. The response is displayed in a clean, formatted interface.
